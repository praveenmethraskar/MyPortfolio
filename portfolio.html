DIAGNOSE ANXIETY AND DEPRESSION IN YOUNG CHILDREN USING ML

  • Duration: 5 Months
  • Language used: Python
  • Project team size: 4 Members

Description

  • This is a Machine Learning project of Diagnosing anxiety and depression in young children. We propose the use of a 90 second ear induction task during the time participant motion is monitoring using available wearable sensors. We extract data and predict diagnosis in a children with and without internalizing diagnosis. The results points towards the future use of this approach in a clinical setting for diagnosing children with internalizing disorders.
